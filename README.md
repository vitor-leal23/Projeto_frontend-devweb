Um projeto simples de um locadora de filmes, onde você tem algumas opções como alugar ou devolver filmes selecionados, ou pesquisar filtrando o filme por genero. 
O projeto também consta com uma Dasboard em tempo real que demosntra quantos filmes estão prontos para locação, quantos ja foram alugados e o total de filmes.
A CineLocadora foi feito apenas por mim Vitor leal dos santos e aqui está um video demosntrando o funcionamento do projeto: https://youtu.be/UzHpiQYPZZI
